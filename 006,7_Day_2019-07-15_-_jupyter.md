# 09 day 2019-07-18

parameter != argument

```python
def func(x):
    return x + 2
```

x == parameter(매개면수)

매개변수는 함수의 정의 부분에서 볼수 있다.

2 == argument(인자, 전달인자)

인자는 함수를 호출하는 부분에서 볼 수 있다.

```python
 func(2)
```
