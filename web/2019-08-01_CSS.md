1. 들여쓰기 2문자

3. 다중 선택 시 한 줄에 선택자를 하나씩 작성

```css
h1, 
h2
{
  color: blue;
}
```

4. 모든 스타일 뒤에는 세미콜론을 붙인다.
5. 스타일 지정 시 , 태그 대신에 클래스를 사용한다(되도록  )
6. 숫자 0이후에는 불필요한 단위를 작성하지 않는다.
7. `@import`  대신 `<link>` 방법을 사용한다.
8. 가능한 한 단축어(축약형) 을 사용한다.
